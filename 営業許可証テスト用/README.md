# 営業許可証テスト用画像の使い方

## ファイル一覧

1. **営業許可証_期限内.html** - 期限内の営業許可証（有効期限: 2025年12月31日）
2. **営業許可証_期限切れ.html** - 期限切れの営業許可証（有効期限: 2023年12月31日）

## 使い方

### 方法1: ブラウザで表示してスクリーンショット

1. HTMLファイルをブラウザで開く
2. ブラウザの開発者ツール（F12）を開く
3. デバイスツールバー（Ctrl+Shift+M / Cmd+Shift+M）でモバイル表示に切り替え（オプション）
4. スクリーンショットを撮る
   - Windows: Win+Shift+S
   - Mac: Cmd+Shift+4
   - または、ブラウザの拡張機能を使用

### 方法2: 印刷してPDF化

1. HTMLファイルをブラウザで開く
2. 印刷（Ctrl+P / Cmd+P）
3. 「PDFに保存」を選択
4. PDFを画像に変換（オンラインツールや画像編集ソフトを使用）

### 方法3: オンラインツールでHTMLを画像化

以下のようなオンラインツールを使用：
- [htmlcsstoimage.com](https://htmlcsstoimage.com/)
- [screenshot.rocks](https://screenshot.rocks/)

## テスト手順

### 期限内の営業許可証のテスト

1. `営業許可証_期限内.html` を画像化
2. 出店者アプリで新規登録
3. 営業許可証として画像をアップロード
4. **期待結果**: 「有効」と表示される（`yes`）

### 期限切れの営業許可証のテスト

1. `営業許可証_期限切れ.html` を画像化
2. 出店者アプリで新規登録
3. 営業許可証として画像をアップロード
4. **期待結果**: 「期限切れ」と表示される（`no`）

## カスタマイズ方法

HTMLファイルを編集して、以下の項目を変更できます：

- **有効期限**: `.expiration-date` の日付を変更
- **営業者名**: `.info-value` の「テスト出店者」を変更
- **営業場所**: `.info-value` の住所を変更
- **許可証番号**: `.license-number` の番号を変更

## 注意事項

- これらのHTMLファイルはテスト用です
- 実際の営業許可証の形式とは異なる場合があります
- AIが正しく読み取れるかどうかは、画像の品質に依存します
- スクリーンショットを撮る際は、十分な解像度を確保してください
