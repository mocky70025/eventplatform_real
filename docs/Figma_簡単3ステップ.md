# FigmaからReactコードを取得：3ステップ

## ステップ1: Figmaで画面を選択
- 実装したい画面をクリック（青い線で囲まれる）

## ステップ2: URLをコピー
- ブラウザのアドレスバーをコピー（Cmd+C / Ctrl+C）

## ステップ3: このチャットに貼り付け
- 貼り付けて送信するだけ！

---

## 例

```
この画面を実装してください
https://www.figma.com/design/XxOyYjBZdBDkaAbqjpbJJ1/kitchen-car?node-id=342-923
```

これで自動的にデザイン情報を取得してReactコードを生成します。
