# 出店者アプリ（store）動作確認チェックリスト

## 確認環境

- **URL**: `https://tomorrow-event-platform-store.vercel.app`
- **ブラウザ**: Chrome/Safari（推奨）
- **デバイス**: PC/スマートフォン

---

## 1. 初期画面・ログイン

### ✅ チェック項目

- [ ] トップページが正しく表示される
- [ ] 「LINEでログイン」ボタンが表示される
- [ ] 「LINEでログイン」ボタンをクリックすると、LINE Login画面に遷移する
- [ ] LINE Loginで認証後、コールバックページに戻ってくる
- [ ] 認証後、メインページにリダイレクトされる

### 🔍 確認ポイント

- ブラウザのコンソール（F12）でエラーが出ていないか確認
- セッションストレージに`line_profile`が保存されているか確認

---

## 2. 出店者登録フォーム

### ✅ チェック項目

#### ステップ1: 基本情報

- [ ] お名前が入力できる
- [ ] 性別が選択できる（男/女/それ以外）
- [ ] 年齢が入力できる
- [ ] 電話番号が入力できる
- [ ] メールアドレスが入力できる
- [ ] ジャンルカテゴリが選択できる
- [ ] ジャンル詳細（自由記述）が入力できる
- [ ] 「次へ」ボタンでステップ2に進める

#### ステップ2: 書類画像アップロード

- [ ] 営業許可証の画像をアップロードできる
- [ ] アップロード後、画像がプレビュー表示される
- [ ] 画像の削除（×ボタン）が動作する
- [ ] 車検証の画像をアップロードできる
- [ ] 自動車検査証の画像をアップロードできる
- [ ] PL保険登録証の画像をアップロードできる
- [ ] 火器類配置図の画像をアップロードできる
- [ ] 各画像が正しく表示される（緑枠で囲まれる）
- [ ] 「次へ」ボタンでステップ3に進める

#### ステップ3: 利用規約

- [ ] 利用規約の内容が表示される
- [ ] 「利用規約を確認する」ボタンで規約ページが開く
- [ ] 利用規約に同意チェックボックスがある
- [ ] チェックを入れないと「登録する」ボタンが押せない
- [ ] 「登録する」ボタンで登録が完了する
- [ ] 登録完了後、適切なメッセージが表示される

### 🔍 確認ポイント

- 画像アップロード時にエラーが出ていないか
- フォームのバリデーションが正しく動作するか
- 必須項目が未入力の場合、エラーメッセージが表示されるか

### ✅ チェック項目

- [ ] イベント一覧が表示される（承認済みイベントのみ）
