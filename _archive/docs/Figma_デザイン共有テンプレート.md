# Figmaデザイン共有テンプレート

## テンプレート1: 1画面の場合

```
この画面を実装してください

【画面名】
Figma URL: https://figma.com/design/[fileKey]/[fileName]?node-id=[nodeId]&m=dev
```

### 例
```
この画面を実装してください

【出店者プロフィール確認画面】
Figma URL: https://figma.com/design/XxOyYjBZdBDkaAbqjpbJJ1/...?node-id=488-473&m=dev
```

## テンプレート2: 複数画面の場合

```

   Figma URL: https://figma.com/design/...?node-id=XXX-XXX&m=dev
```

## テンプレート3: カテゴリ別に整理

```
【カテゴリ名】の画面を実装してください

### 認証関連
- ログイン画面: https://figma.com/design/...?node-id=488-465&m=dev
- 登録画面: https://figma.com/design/...?node-id=488-466&m=dev
- メール確認画面: https://figma.com/design/...?node-id=488-467&m=dev

### プロフィール関連
- プロフィール確認画面: https://figma.com/design/...?node-id=488-473&m=dev
- プロフィール編集画面: https://figma.com/design/...?node-id=488-476&m=dev
```

---

## 重要なポイント

### ✅ 必ず含めるもの

1. **Figma URL**
   - `node-id` が含まれていること
   - `&m=dev` を付けるとより詳細な情報が取得できます（推奨）

2. **画面名・説明**
   - どの画面かわかるように

### ✅ 推奨事項

- 関連する画面はまとめて提供（一度に実装できる）
- リスト形式で整理するとわかりやすい

### ❌ 避けるべきこと

- URLに `node-id` がない
- 50画面を一度に（1画面ずつ、または関連画面をまとめて）

---

## 使い方

1. このテンプレートをコピー
2. 画面名とFigma URLを入力
3. チャットに貼り付け

これで完全に実装できます！
