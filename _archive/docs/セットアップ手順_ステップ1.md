# セットアップ手順 - ステップ1: Supabaseプロジェクトの作成

## 🎯 このステップの目標
**Supabaseプロジェクトを作成する**

---

## 📋 手順（超詳細版）

### ステップ1-1: Supabaseにアクセス

1. **ブラウザを開く**
   - Chrome、Safari、Firefoxなど、お好みのブラウザを使用

2. **SupabaseのURLにアクセス**
   - 以下のURLをブラウザのアドレスバーに入力：
   ```
   https://supabase.com
   ```

3. **Enterキーを押す**
   - ページが読み込まれるまで待つ

---

### ステップ1-2: ログイン（または新規登録）

#### 既にSupabaseアカウントがある場合

1. **「Sign In」ボタンをクリック**
   - ページ右上の「Sign In」ボタンを探す
   - クリックする

2. **ログイン情報を入力**
   - メールアドレスを入力
   - パスワードを入力

3. **「Sign In」ボタンをクリック**
   - ログインが完了するまで待つ

#### Supabaseアカウントがない場合

1. **「Start your project」ボタンをクリック**
   - ページの中央または右上にあるボタンを探す
   - クリックする

2. **新規登録フォームを入力**
   - メールアドレスを入力
   - パスワードを入力（8文字以上）
   - 利用規約に同意するチェックボックスにチェックを入れる

3. **「Create account」ボタンをクリック**
   - メール確認が必要な場合、メールを確認して認証する

---

### ステップ1-3: ダッシュボードに移動

1. **ログイン後、自動的にダッシュボードに移動**
   - もし移動しない場合は、以下のURLにアクセス：
   ```
   https://supabase.com/dashboard
   ```

2. **ダッシュボードが表示されることを確認**
   - 左側にメニューがあることを確認
   - 「Projects」という項目があることを確認

---

### ステップ1-4: 新しいプロジェクトを作成

1. **「New Project」ボタンを探す**
   - ダッシュボードの右上または中央に「New Project」ボタンがある
   - クリックする

2. **プロジェクト情報を入力**

   #### 2-1. Organization（組織）を選択
   - ドロップダウンメニューから組織を選択
   - 個人の場合は「Personal」を選択

   #### 2-2. Name（プロジェクト名）を入力
   - プロジェクト名を入力
   - 例: `tomorrow-event-platform`
   - または: `tomorrow`
   - または: お好みの名前

   #### 2-3. Database Password（データベースパスワード）を設定
   - パスワードを入力（12文字以上推奨）
   - **重要**: このパスワードは後で必要になるので、安全な場所にメモしておく
   - 例: `MySecurePassword123!`

   #### 2-4. Region（リージョン）を選択
   - ドロップダウンメニューからリージョンを選択
   - 日本からアクセスする場合は「Northeast Asia (Tokyo)」を推奨
   - または、お好みのリージョンを選択

   #### 2-5. Pricing Plan（料金プラン）を選択
   - 「Free」プランを選択（無料プラン）
   - または、有料プランを選択（必要に応じて）

3. **「Create new project」ボタンをクリック**
   - プロジェクトの作成が開始される
   - 数分かかる場合があるので、待つ

---

### ステップ1-5: プロジェクトの作成完了を待つ

1. **プロジェクト作成中の画面が表示される**
   - 「Setting up your project...」などのメッセージが表示される
   - プログレスバーが表示される場合がある

2. **完了するまで待つ**
   - 通常、1〜3分程度かかる
   - 「Your project is ready!」などのメッセージが表示されるまで待つ

3. **自動的にプロジェクトページに移動**
   - プロジェクトのダッシュボードが表示される

---

### ステップ1-6: プロジェクト情報を確認

1. **左側のメニューを確認**
   - 以下のメニュー項目があることを確認：
     - Table Editor
     - SQL Editor
     - Authentication
     - Storage
     - Settings
     - など

2. **プロジェクトURLを確認**
   - 左側のメニューから「Settings」をクリック
   - 「API」をクリック
   - 「Project URL」を確認
   - 例: `https://xxxxxxxxxxxxx.supabase.co`
   - **重要**: このURLをメモしておく

3. **API Keyを確認**
   - 同じ画面で「Project API keys」セクションを確認
   - 「anon public」キーを確認
   - **重要**: このキーをメモしておく（表示されている場合はコピー）

---

### ステップ1-7: 情報をメモする

以下の情報を安全な場所にメモしてください：

1. **Project URL**
   - 例: `https://xxxxxxxxxxxxx.supabase.co`

2. **anon public key**
   - 例: `eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`（長い文字列）

3. **Database Password**
   - ステップ1-4で設定したパスワード

## ⚠️ トラブルシューティング

### プロジェクトが作成されない場合

1. **ブラウザをリフレッシュ**
   - F5キーまたはCmd+R（Mac）/ Ctrl+R（Windows）

2. **別のブラウザで試す**
   - Chrome、Safari、Firefoxなど

3. **時間をおいて再試行**
   - 数分待ってから再度試す

### ログインできない場合

1. **パスワードをリセット**
   - 「Forgot password?」リンクをクリック

2. **メールアドレスを確認**
   - 正しいメールアドレスを入力しているか確認
