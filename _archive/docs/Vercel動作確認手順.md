## 概要

4つのプロジェクトの動作確認を行います。

---

## プロジェクト1: store-web（出店者アプリ - Web環境）

**URL**: https://store-web-ten.vercel.app/

### 確認項目

#### 1. 初期画面の表示
- [ ] アプリが正常に表示される
- [ ] 「ログイン」と「新規登録」のボタンが表示される
- [ ] **重要**: LINE認証のボタンが表示されない（メール認証のみ）

#### 2. メール認証での新規登録
- [ ] 「新規登録」をクリック

- [ ] メールアドレスとパスワードを入力
- [ ] 新規登録を実行
- [ ] メール確認待ち画面が表示される（またはメールが送信される）
- [ ] メール確認後、登録フォームが表示される

#### 3. メール認証でのログイン
- [ ] 「ログイン」をクリック

- [ ] 登録済みのメールアドレスとパスワードを入力
- [ ] ログインを実行
- [ ] ホーム画面（イベント一覧）が表示される

#### 4. 基本機能の確認
- [ ] イベント一覧が表示される
- [ ] イベント詳細が表示される
- [ ] イベントへの申し込みができる

## トラブルシューティング

### アプリが表示されない

1. Vercel Dashboardでデプロイ状態を確認
2. ビルドログを確認してエラーがないか確認
3. 環境変数が正しく設定されているか確認

### 認証が動作しない

1. 環境変数が正しく設定されているか確認
2. Supabase Dashboardで認証設定を確認
3. ブラウザのコンソールでエラーメッセージを確認

### LINE認証が動作しない（store-liff）

1. LINE Developers ConsoleのCallback URLが正しく設定されているか確認
2. LIFFアプリのEndpoint URLが正しく設定されているか確認
3. 環境変数 `NEXT_PUBLIC_LINE_LOGIN_CHANNEL_ID` と `LINE_LOGIN_CHANNEL_SECRET` が正しいか確認

- [ ] 認証フローが正常に動作する
- [ ] 基本機能が動作する
- [ ] エラーが発生しない
