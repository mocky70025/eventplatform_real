# Figmaエフェクトスタイル完全リスト

## 登録用チェックリスト

各エフェクトスタイル（シャドウ）をFigmaで登録する際の完全な情報です。

---

## 1. Effect/Shadow Level 1

- **名前**: `Effect/Shadow Level 1`
- **タイプ**: `Drop shadow`
- **X**: `0px`
- **Y**: `2px`
- **Blur**: `8px`
- **Spread**: `0px`
- **カラー**: `rgba(0, 0, 0, 0.1)` または `#000000` の不透明度10%
- **説明**: カード用のシャドウ

**Figmaでの設定方法:**
1. オブジェクトを選択
2. 右側のパネルで「Effects」セクションを開く
3. 「+」をクリックして「Drop shadow」を追加
4. 以下の値を設定：
   - **X**: `0`
   - **Y**: `2`
   - **Blur**: `8`
   - **Spread**: `0`
   - **カラー**: 黒色、不透明度 `10%`
5. 「Effect styles」で「+」をクリック
6. スタイル名: `Effect/Shadow Level 1` を入力
7. 「スタイルの作成」をクリック
