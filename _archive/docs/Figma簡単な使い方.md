# FigmaからReactコードを取得する超簡単ガイド

## 一番簡単な方法

**Figmaで要素を選択 → URLをコピー → このチャットに貼り付け**

それだけです！

---

## ステップバイステップ

### ステップ1: Figmaで要素を選択

1. Figmaファイルを開く
2. 実装したい画面やコンポーネントをクリックして選択
   - 画面全体を選択したい場合 → 画面全体を囲むフレームをクリック
   - ボタンだけを選択したい場合 → そのボタンをクリック

### ステップ2: URLをコピー

1. ブラウザのアドレスバーをクリック（`Cmd + L` / `Ctrl + L`）
2. 全選択（`Cmd + A` / `Ctrl + A`）
3. コピー（`Cmd + C` / `Ctrl + C`）

**例**:
```
https://www.figma.com/design/XxOyYjBZdBDkaAbqjpbJJ1/kitchen-car?node-id=342-923
```

### ステップ3: このチャットに貼り付け

1. このチャットの入力欄に貼り付け（`Cmd + V` / `Ctrl + V`）
2. 送信ボタンをクリック

**これで完了！** 自動的にデザイン情報を取得して、Reactコードを生成します。

## よくある質問

### Q1: どの要素を選択すればいいの？

**A: 実装したい画面全体を囲むフレームを選択してください**

- ✅ **良い例**: 画面全体を囲むフレームを選択
- ❌ **悪い例**: ボタンだけ、テキストだけを選択

**確認方法**:
- Figmaで選択した要素の周りに青い線が表示されます
- その青い線で囲まれた範囲が取得されます

### Q2: 画面全体が選択できません

**A: 以下の方法を試してください**

1. **レイヤーパネルから選択**
   - 左側のレイヤーパネルを開く
   - 画面全体を囲むフレームを見つける
   - クリックして選択

2. **親フレームを探す**
   - 選択した要素を右クリック
   - 「Select Parent」を選択
   - 画面全体を囲むフレームまで繰り返す

### Q3: URLに`node-id`がありません

**A: 要素を選択してからURLをコピーしてください**

- 要素を選択すると、URLに自動的に`?node-id=...`が追加されます
- `node-id`がない場合は、何も選択されていません

### Q4: 複数の画面を一度に取得したい

**A: 1つずつ取得してください**

1. 1つ目の画面を選択 → URLをコピー → このチャットに貼り付け
2. 2つ目の画面を選択 → URLをコピー → このチャットに貼り付け
3. 繰り返す

**または**、複数の画面が1つの親フレーム内にある場合は、親フレームを選択すれば全て取得できます。

## 実用的なワークフロー

### ワークフロー1: 1画面ずつ実装する場合

```
1. Figmaで画面を選択
2. URLをコピー
3. このチャットに「この画面を実装してください」とURLを貼り付け
4. 生成されたコードを確認・実装
5. 次の画面に進む
```

### ワークフロー2: 全体を確認してから実装する場合

```
1. 各画面のURLを全て集める
2. このチャットに全てのURLを貼り付けて「全ての画面の情報を取得してください」
3. デザイン仕様書を確認
4. 1つずつ実装していく
```

## トラブルシューティング

### 問題1: URLを貼り付けたけどエラーが出る

**解決方法**:
- URLが正しくコピーされているか確認（`node-id=...`が含まれているか）
- Figmaファイルにアクセス権限があるか確認
- URLが途中で切れていないか確認

### 問題2: 取得した情報が違う

**解決方法**:
- Figmaで正しい要素が選択されているか確認
- もう一度URLをコピーして貼り付け

### 問題3: 一部の要素が取得されない

**解決方法**:
- より大きな親フレームを選択してみる
- または、その要素だけを個別に選択して取得

## より簡単な代替方法

URLが使いづらい場合は、以下の方法もあります：

### 方法1: スクリーンショットを共有

1. Figmaでスクリーンショットを撮る
2. このチャットに画像として送信
3. 画像からデザイン情報を読み取る（精度は下がります）

### 方法2: 手動で情報を共有

1. Figmaで要素を選択
2. 右パネルから以下の情報をコピー：
   - 位置、サイズ
   - 色（HEX値）
   - フォント情報
3. このチャットに情報を貼り付け

**最も簡単な方法**:
1. Figmaで要素を選択
2. URLをコピー
3. このチャットに貼り付け
