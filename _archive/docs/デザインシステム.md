# デミセル デザインシステム

## 1. カラーパレット

### プライマリカラー（メインカラー）
**LINEグリーン**をベースにしたカラースキーム

- **Primary Green** (`#06C755`)
  - 用途: メインボタン、リンク、強調表示、承認済みステータス
  - 説明: LINEのブランドカラーと同じ緑色。信頼感と親しみやすさを表現
  - 使用例: 「ログイン」ボタン、「申し込む」ボタン、「承認済み」バッジ

- **Primary Green Dark** (`#05B04A`)
  - 用途: ボタンのホバー状態
  - 説明: プライマリグリーンを少し濃くした色
  - 使用例: ボタンにマウスを乗せた時

- **Primary Green Light** (`#E6F7ED`)
  - 用途: 成功メッセージの背景、承認済みステータスの背景
  - 説明: プライマリグリーンを薄くした淡い緑色
  - 使用例: 「承認済み」バッジの背景、「成功しました」メッセージの背景

### ニュートラルカラー（グレースケール）
テキストや背景に使用する基本色

- **Black** (`#000000`) - 真っ黒
  - 用途: メインテキスト、見出し
  - 説明: 最も濃い黒。主要なテキストに使用

- **Gray 900** (`#1A1A1A`) - ほぼ黒
  - 用途: 見出し、重要なテキスト
  - 説明: 黒に近い濃いグレー

- **Gray 700** (`#333333`) - 濃いグレー
  - 用途: 本文テキスト
  - 説明: 読みやすい濃いグレー

- **Gray 500** (`#666666`) - 中間グレー
  - 用途: サブテキスト、補足情報
  - 説明: 本文より目立たない情報に使用

- **Gray 300** (`#999999`) - 薄いグレー
  - 用途: プレースホルダー、無効な要素
  - 説明: 入力欄のプレースホルダーなどに使用

- **Gray 200** (`#E5E5E5`) - とても薄いグレー
  - 用途: ボーダー、区切り線
  - 説明: 要素を区切る線に使用

- **Gray 100** (`#F7F7F7`) - ほぼ白のグレー
  - 用途: ページ背景、セクション背景
  - 説明: 白より少しグレーがかった背景色

- **White** (`#FFFFFF`) - 真っ白
  - 用途: カード背景、モーダル背景
  - 説明: コンテンツを配置する白い背景

### ステータスカラー（状態を表す色）
操作の結果や状態を視覚的に伝える色

- **Success** (`#06C755`) - 緑
  - 用途: 成功メッセージ、承認済みステータス
  - 説明: プライマリグリーンと同じ。成功や承認を表す
  - 使用例: 「承認済み」バッジ、「保存しました」メッセージ

- **Success Light** (`#E6F7ED`) - 薄い緑
  - 用途: 成功メッセージの背景
  - 説明: 成功を表す背景色

- **Warning** (`#FFB800`) - オレンジ/黄色
  - 用途: 警告メッセージ、審査中ステータス
  - 説明: 注意が必要な状態を表す
  - 使用例: 「審査中」バッジ、「確認が必要です」メッセージ

- **Warning Light** (`#FFF9E6`) - 薄い黄色
  - 用途: 警告メッセージの背景、審査中ステータスの背景
  - 説明: 警告を表す背景色

- **Error** (`#FF3B30`) - 赤
  - 用途: エラーメッセージ、却下ステータス
  - 説明: エラーや却下を表す
  - 使用例: 「却下」バッジ、「エラーが発生しました」メッセージ

- **Error Light** (`#FFEBEE`) - 薄い赤
  - 用途: エラーメッセージの背景
  - 説明: エラーを表す背景色

- **Info** (`#0066FF`) - 青
  - 用途: 情報メッセージ
  - 説明: 補足情報を表す
  - 使用例: 「お知らせ」バナー

- **Info Light** (`#E6F0FF`) - 薄い青
  - 用途: 情報メッセージの背景
  - 説明: 情報を表す背景色

### セマンティックカラー（意味を持った色の組み合わせ）
特定の意味を持つ色の組み合わせ

- **Pending（審査中）**
  - 背景: `#FFF9E6` - 薄い黄色
  - テキスト: `#B8860B` - 濃い黄色/ゴールド
  - 使用例: 「審査中」バッジ

- **Approved（承認済み）**
  - 背景: `#E6F7ED` - 薄い緑
  - テキスト: `#06C755` - 緑
  - 使用例: 「承認済み」バッジ

- **Rejected（却下）**
  - 背景: `#FFE6E6` - 薄い赤
  - テキスト: `#FF3B30` - 赤
  - 使用例: 「却下」バッジ
