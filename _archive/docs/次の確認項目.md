# 次の確認項目

## 優先度：高（すぐに確認すべき）

### 1. イベント一覧の表示確認 ✅
- [ ] テスト用イベントデータをSQLで追加済み
- [ ] 出店者アプリでイベント一覧が表示される
- [ ] イベントカードが正しく表示される（イベント名、日付、会場など）
- [ ] エラーメッセージが出ていない

**確認方法**:
1. 出店者アプリにログイン
2. イベント一覧ページを開く
3. 5つのテストイベントが表示されるか確認

---

### 2. イベント詳細・申し込み機能の確認
- [ ] イベントカードをクリックすると詳細ページが表示される
- [ ] イベント詳細情報が正しく表示される（開催期間、会場、説明など）
- [ ] 「出店申し込み」ボタンが表示される
- [ ] 「出店申し込み」ボタンをクリックすると申し込みが完了する
- [ ] 申し込み完了メッセージが表示される

**確認方法**:
1. イベント一覧からイベントを選択
2. 詳細ページを確認
3. 「出店申し込み」ボタンをクリック
4. 申し込みが成功するか確認

---

### 3. 申し込み管理ページの確認
- [ ] 「申し込み管理」タブをクリックすると申し込み一覧が表示される
- [ ] 申し込んだイベントが一覧に表示される
- [ ] 申し込みステータスが正しく表示される（審査中/承認/却下）
- [ ] イベント情報が正しく表示される

**確認方法**:
1. ナビゲーションバーから「申し込み管理」を選択
2. 申し込み一覧を確認
3. ステータス表示を確認
