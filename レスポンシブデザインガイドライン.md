# レスポンシブデザインガイドライン

## ブレークポイント

### デバイス別の定義
- **Mobile（スマートフォン）**: 0px - 767px
  - 主なデバイス: iPhone, Android
  - レイアウト: シングルカラム、ボトムナビゲーション
  - 最大幅: 394px（コンテンツエリア）

- **Tablet（タブレット）**: 768px - 1023px
  - 主なデバイス: iPad, Android Tablet
  - レイアウト: 2カラム（オプション）、サイドナビゲーション
  - 最大幅: 768px（コンテンツエリア）

- **Desktop（デスクトップ）**: 1024px+
  - 主なデバイス: PC, ラップトップ
  - レイアウト: マルチカラム、サイドバー、グリッドレイアウト
  - 最大幅: 1200px（コンテンツエリア）

---

## レイアウトの違い

### ナビゲーション

#### Mobile（スマートフォン）
- **位置**: 画面下部（ボトムナビゲーション）
- **スタイル**: 固定、アイコン + ラベル
- **高さ**: 64px
- **アイコンサイズ**: 22x22px
- **フォントサイズ**: 12px

#### Tablet（タブレット）
- **位置**: 画面左側（サイドバー）または上部（トップバー）
- **スタイル**: 折りたたみ可能、アイコン + ラベル
- **幅**: 240px（サイドバーの場合）
- **アイコンサイズ**: 24x24px
- **フォントサイズ**: 14px

#### Desktop（デスクトップ）
- **位置**: 画面左側（サイドバー）または上部（トップバー）
- **スタイル**: 常時表示、アイコン + ラベル
- **幅**: 280px（サイドバーの場合）
- **アイコンサイズ**: 24x24px
- **フォントサイズ**: 16px

## 実装方法

### CSS Media Queries

```css
/* Mobile First */
.container {
  max-width: 394px;
  padding: 0 16px;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    max-width: 768px;
    padding: 0 24px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
    padding: 0 32px;
  }
}
```

### Tailwind CSS ブレークポイント

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    screens: {
      'sm': '640px',
      'md': '768px',
      'lg': '1024px',
      'xl': '1280px',
    },
  },
}
```

### Next.js Image コンポーネント

```tsx
import Image from 'next/image'

<Image
  src="/assets/logo.svg"
  alt="Logo"
  width={200}
  height={60}
  className="w-[160px] h-[48px] md:w-[200px] md:h-[60px] lg:w-[240px] lg:h-[72px]"
/>
```

## パフォーマンス最適化
