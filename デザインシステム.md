# Tomorrow Event Platform デザインシステム

## 1. カラーパレット

### プライマリカラー
- **Primary Green**: `#06C755` - メインアクション、リンク、強調
- **Primary Green Dark**: `#05B04A` - ホバー状態
- **Primary Green Light**: `#E6F7ED` - 背景、成功メッセージ

### ニュートラルカラー
- **Black**: `#000000` - メインテキスト
- **Gray 900**: `#1A1A1A` - 見出し
- **Gray 700**: `#333333` - 本文
- **Gray 500**: `#666666` - サブテキスト
- **Gray 300**: `#999999` - プレースホルダー
- **Gray 200**: `#E5E5E5` - ボーダー、区切り線
- **Gray 100**: `#F7F7F7` - 背景
- **White**: `#FFFFFF` - カード、背景

### ステータスカラー
- **Success**: `#06C755` - 成功、承認済み
- **Success Light**: `#E6F7ED` - 成功背景
- **Warning**: `#FFB800` - 警告、審査中
- **Warning Light**: `#FFF9E6` - 警告背景
- **Error**: `#FF3B30` - エラー、却下
- **Error Light**: `#FFEBEE` - エラー背景
- **Info**: `#0066FF` - 情報
- **Info Light**: `#E6F0FF` - 情報背景

### セマンティックカラー
- **Pending**: `#FFF9E6` (背景) + `#B8860B` (テキスト) - 審査中
- **Approved**: `#E6F7ED` (背景) + `#06C755` (テキスト) - 承認済み
- **Rejected**: `#FFE6E6` (背景) + `#FF3B30` (テキスト) - 却下
