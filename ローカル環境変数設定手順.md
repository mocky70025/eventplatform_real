# ローカル環境変数設定手順

## 必要な環境変数

### 1. store/.env.local を作成

```bash
cd store
touch .env.local
```

以下の内容を設定：

```env
# Supabase設定
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# LINE Login設定
NEXT_PUBLIC_LINE_LOGIN_CHANNEL_ID=your_line_login_channel_id
LINE_LOGIN_CHANNEL_SECRET=your_line_login_channel_secret

# ローカル開発用
NEXT_PUBLIC_LINE_LOGIN_REDIRECT_URI=http://localhost:3000/auth/callback
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 2. organizer/.env.local を作成

```bash
cd organizer
touch .env.local
```

以下の内容を設定：

```env
# Supabase設定
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key

# LINE Login設定
NEXT_PUBLIC_LINE_LOGIN_CHANNEL_ID=your_line_login_channel_id
LINE_LOGIN_CHANNEL_SECRET=your_line_login_channel_secret

# ローカル開発用
NEXT_PUBLIC_LINE_LOGIN_REDIRECT_URI=http://localhost:3001/auth/callback
NEXT_PUBLIC_APP_URL=http://localhost:3001
```

## 環境変数の取得方法

### Supabase設定
1. Supabase Dashboardにアクセス
2. Settings > API から以下を取得：
   - `NEXT_PUBLIC_SUPABASE_URL`: Project URL
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`: anon public key

### LINE Login設定
1. LINE Developers Consoleにアクセス
2. チャネル設定から以下を取得：
   - `NEXT_PUBLIC_LINE_LOGIN_CHANNEL_ID`: Channel ID
   - `LINE_LOGIN_CHANNEL_SECRET`: Channel secret
3. コールバックURLを設定：
   - `http://localhost:3000/auth/callback` (store用)
   - `http://localhost:3001/auth/callback` (organizer用)

## ローカルで起動

### store（出店者用）
```bash
cd store
npm run dev
```
→ `http://localhost:3000` でアクセス

### organizer（主催者用）
```bash
cd organizer
npm run dev
```
→ `http://localhost:3001` でアクセス

## 注意事項

- `.env.local` ファイルは `.gitignore` に含まれているため、Gitにはコミットされません
- 環境変数を変更した場合は、開発サーバーを再起動してください
- Vercelの環境変数と異なる場合は、ローカル用の値を設定してください
