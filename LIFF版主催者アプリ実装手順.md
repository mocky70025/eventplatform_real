# LIFF版主催者アプリ実装手順

## 概要

LIFF版の主催者アプリで、メールアドレス認証とLINE認証の両方に対応します。

## 実装方針

1. **LIFF環境の検出**: LIFF環境かどうかを検出
2. **認証方法**: 
   - LINE認証: LIFF環境でもLINE Login（OAuth）を使用（web版と同じ）
   - メールアドレス認証: LIFF環境でも動作するように実装
3. **UI/UX**: LIFF環境でもweb版と同じUIを使用

## 実装手順

### 1. LIFF SDKのインストール（オプション）

LIFF環境の検出のみが必要な場合は、SDKは不要です。
ただし、LIFF環境でのみ利用可能な機能を使用する場合は、SDKが必要です。

```bash
cd organizer
npm install @line/liff
```

### 2. LIFF環境の検出

LIFF環境かどうかを検出する関数を追加します。

### 3. 認証フローの実装

- LINE認証: web版と同じ（LINE Login OAuth）
- メールアドレス認証: LIFF環境でも動作するように実装

### 4. 動作確認

LIFF環境での動作確認を行います。

## 注意点

- LIFF環境では、外部ブラウザへのリダイレクトが制限される場合があります
- メール確認リンクは、外部ブラウザで開く必要がある場合があります
- LINE Login（OAuth）は、LIFF環境でも動作しますが、外部ブラウザへのリダイレクトが必要です
