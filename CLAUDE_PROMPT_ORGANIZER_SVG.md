# Claude用プロンプト：主催者アプリ SVGデザイン生成

## 背景情報

主催者アプリ（モバイル版）のSVGデザインを作成しています。出店者アプリと基本構造は同じですが、カラーパレットとフォーム内容が異なります。

## デザインシステム（カラーパレット）

以下のカラーパレットを厳密に使用してください：

- **Primary**: `#FF8A5C`
- **Secondary**: `#FF8A5C`（主催者アプリのメインカラー - オレンジ）
- **Background**: `#E8F5F5`（薄い青緑 - 出店者アプリの`#FFF5F0`（薄いオレンジ）と逆の色）
- **Shadow**: 8% opacity black（`rgba(0, 0, 0, 0.08)`）

### その他の使用色

- **テキスト色（メイン）**: `#2C3E50`
- **テキスト色（サブ）**: `#6C757D`
- **ボーダー色**: `#E9ECEF`
- **白**: `#FFFFFF`
- **グレー（ナビゲーション非アクティブ）**: `#666666`
- **Googleボタン**: `#FFFFFF`（白背景、ボーダー付き）

**注意**: 主催者アプリにはLINEログインはありません。Googleとメールアドレスのみです。

## 画面サイズ

- **幅**: 393px
- **高さ**: 852px
- **ViewBox**: `0 0 393 852`

## 主催者アプリの特徴

### 認証方法
- **Googleログイン**: あり
- **メールアドレス/パスワード**: あり
- **LINEログイン**: なし（出店者アプリとの違い）

### 登録フォーム（RegistrationForm）のフィールド

主催者アプリの登録フォームは以下のフィールドを含みます：

**ステップ1 - 基本情報入力**:
- 会社名（company_name）※必須
- お名前（name）※必須
- 性別（gender）※必須 - 「男」「女」「それ以外」から選択
- 年齢（age）※必須
- 電話番号（phone_number）※必須
- メールアドレス（email）※必須

**注意**: 出店者アプリと異なり、書類アップロード（営業許可証、車検証など）はありません。

**ステップ2 - 情報確認**:
- 入力した情報の確認リスト

**ステップ3 - 登録完了**:
- 登録完了メッセージ

### イベント管理機能

主催者アプリには以下の機能があります：

1. **イベント作成（EventForm）**: 詳細なイベント情報を入力
2. **イベント管理（EventManagement）**: 作成したイベントの一覧と管理
3. **申し込み管理（EventApplications）**: 出店者からの申し込みを管理
4. **プロフィール管理（OrganizerProfile）**: 主催者情報の表示・編集

## SVG作成のガイドライン

1. **構造**:
   - `<svg>`タグで開始
   - `<g clip-path="url(#clip0_X)">`でコンテンツをラップ
   - `<defs>`セクションでフィルターとクリップパスを定義

2. **シャドウ**:
   - カードやボタンには`filter`要素を使用
   - `feOffset dy="8"`、`feGaussianBlur stdDeviation="16"`、`feColorMatrix`で8% opacity blackを設定

3. **角丸**:
   - `rx`属性を使用（通常8px、カードは12px、ボタンは12px）

4. **テキスト**:
   - `font-family="Arial"`を使用
   - フォントサイズ: タイトル18px、本文14px、サブテキスト12px
   - フォントウェイト: タイトル600、本文400

5. **アイコン**:
   - SVGパスで描画するか、シンプルな図形で表現
   - ナビゲーションアイコンは24x24px程度

6. **一貫性**:
   - 出店者アプリのデザインパターンを踏襲（色のみ変更）
   - 同じスタイルのボタン、入力フィールド、カードを使用
   - **主催者アプリのメインカラーは`#FF8A5C`（オレンジ）を使用。背景は`#E8F5F5`（薄い青緑）**

7. **色の適用箇所**:
   - ロゴ背景: `#FF8A5C`（Secondary - オレンジ）
   - アクティブタブの下線: `#FF8A5C`（Secondary - オレンジ）
   - ボタン背景: `#FF8A5C`（Secondary - オレンジ）
   - プログレスインジケーター: `#FF8A5C`（Secondary - オレンジ）
   - フォーカス時のボーダー: `#FF8A5C`（Secondary - オレンジ）
   - ナビゲーションアクティブ: `#FF8A5C`（Secondary - オレンジ）
   - アイコン背景: `#FF8A5C`（Secondary - オレンジ）
   - 画面背景: `#E8F5F5`（薄い青緑）

各画面について、以下の形式でSVGコードを出力してください：

## 注意事項

- **主催者アプリにはLINEログインはありません**。Googleとメールアドレスのみです。
- すべての色はデザインシステムに従ってください（`#FF8A5C`（Secondary）をメインカラーとして使用。背景は`#E8F5F5`（薄い青緑）です。出店者アプリとは色が逆になります）。
- モバイルファーストのデザインを意識してください。
- 実用的で使いやすいUIを心がけてください。

- 登録フォームには書類アップロードはありません（出店者アプリとの違い）。
- イベント作成フォームは3ステップに分かれています。
