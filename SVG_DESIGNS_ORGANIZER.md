# 主催者アプリ SVGデザインストア

このファイルには、主催者アプリのUIデザイン（SVG形式）を保存します。

## デザインシステム

### カラーパレット

- **Primary**: #FF8A5C
- **Secondary**: #FF8A5C（主催者アプリのメインカラー - オレンジ）
- **Background**: #E8F5F5（薄い青緑 - 出店者アプリの`#FFF5F0`（薄いオレンジ）と逆の色）
- **Shadow**: 8% opacity black（`rgba(0, 0, 0, 0.08)`）

### その他の使用色

- **テキスト色（メイン）**: #2C3E50
- **テキスト色（サブ）**: #6C757D
- **ボーダー色**: #E9ECEF
- **白**: #FFFFFF
- **グレー（ナビゲーション非アクティブ）**: #666666
- **Googleボタン**: #FFFFFF（白背景、ボーダー付き）

## 画面サイズ

- **幅**: 393px
- **高さ**: 852px
- **ViewBox**: `0 0 393 852`

## 主催者アプリの特徴

### 認証方法
- **Googleログイン**: あり
- **メールアドレス/パスワード**: あり
- **LINEログイン**: なし（出店者アプリとの違い）

### 登録フォーム（RegistrationForm）のフィールド

**ステップ1 - 基本情報入力**:
- 会社名（company_name）※必須
- お名前（name）※必須
- 性別（gender）※必須 - 「男」「女」「それ以外」から選択
- 年齢（age）※必須
- 電話番号（phone_number）※必須
- メールアドレス（email）※必須

**注意**: 出店者アプリと異なり、書類アップロード（営業許可証、車検証など）はありません。

**ステップ2 - 情報確認**:
- 入力した情報の確認リスト

**ステップ3 - 登録完了**:
- 登録完了メッセージ

## 実装時の注意事項

1. **主催者アプリにはLINEログインはありません**。Googleとメールアドレスのみです。
2. すべての色はデザインシステムに従ってください（`#FF8A5C`（Secondary）をメインカラーとして使用。背景は`#E8F5F5`（薄い青緑）です。出店者アプリとは色が逆になります）。
3. モバイルファーストのデザインを意識してください。
4. 実用的で使いやすいUIを心がけてください。
5. 出店者アプリのデザインパターンとの一貫性を保ちつつ、色のみ変更してください。
6. 登録フォームには書類アップロードはありません（出店者アプリとの違い）。
7. イベント作成フォームは複数ステップに分かれています。
