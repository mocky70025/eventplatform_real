# 動作確認チェックリスト（優先順位順）

## 🔴 最優先：最近修正した機能

### 1. 主催者承認機能（管理者アプリ）
**URL**: http://localhost:3000/organizers

- [ ] 主催者一覧が表示される
- [ ] 「承認する」ボタンをクリック
- [ ] ステータスが「未承認」→「承認済み」に変わる
- [ ] ページをリロードしても状態が維持される
- [ ] Supabaseで実際に`is_approved`が`true`になっていることを確認
- [ ] 「承認解除」ボタンで元に戻せる

### 2. 主催者アプリの承認状態表示
**URL**: http://localhost:3002

- [ ] **未承認の主催者でログイン**
  - [ ] 承認待ちバナー（黄色）が表示される
  - [ ] 「新しいイベントを作成」ボタンが無効化されている
  - [ ] イベント一覧が空の場合、「承認待ちです」メッセージが表示される
  - [ ] イベント作成ページにアクセスすると警告が表示される

- [ ] **承認済みの主催者でログイン**
  - [ ] 承認済みバナー（緑色）が表示される
  - [ ] 「新しいイベントを作成」ボタンがクリック可能
  - [ ] イベント作成ページにアクセスできる

### 3. プロフィール更新（出店者アプリ）
**URL**: http://localhost:3001/profile

- [ ] プロフィールページが表示される
- [ ] 既存のプロフィール情報が表示される
- [ ] **営業許可証等の書類画像が表示される**（重要）
- [ ] プロフィール情報を変更して保存
- [ ] エラーが発生しない
- [ ] 変更が保存される
